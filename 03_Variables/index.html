Why don't you need to use the var keyword every time you use a variable (right now, we only use it when we create the variable)? What does var do?

As I understand it, var sets/declares a variable (var a). Once it is set, it can be assigned/reassigned at will without another var.

Try removing the keyword var from any line where it might show up in your program. What happens? Does your program still work as intended? Do any error messages show up in the console? If so, what error message?

There is no error message because the var isn't technically needed when using variables on the global scope. However, not using var when declaring a function variable makes that variable global in scope, so it's always advised to use var

What is the difference between a string and an integer? What kinds of information can you use in each? Why might you use one instead of the other?

string is a non-consequential sequence of characters, while a number(integer or float) has a mathmatical value, however the two appear to be more interchangable than in other languages. Choosing a type to use depends on what type functions you want access to most likely.
